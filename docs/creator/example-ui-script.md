# UI와 스크립트 연동 실습

이번 실습에서는 **버튼을 누르면 대사가 재생되는 인터랙션**을 만들어봅니다.

---

## 1️⃣ UI 버튼 추가

1. **UI 탭**을 엽니다.  
2. “+ 버튼” 아이콘을 눌러 새 버튼을 추가합니다.  
3. 텍스트를 “재생”으로 설정합니다.

---

## 2️⃣ 이벤트 연결

- 버튼을 선택하고 오른쪽 속성 창에서  
  “클릭 시 실행할 스크립트”를 선택합니다.  
- 새 스크립트를 만들고 다음 내용을 입력합니다.

    사운드 [Voice1]을(를) 재생합니다.  
    자막 [Line1]을(를) 표시합니다.

---

## 3️⃣ 실행 테스트

- 플레이 모드로 전환합니다.  
- “재생” 버튼을 누르면 지정된 사운드와 자막이 동시에 재생됩니다.
